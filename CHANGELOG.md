# Changelog

All notable changes to CareOS Healthcare Portal.

## [1.4.0] — December 2024
### Added
- AI Benefits Copilot with streaming response UX
- Confidence indicator on AI responses
- Human escalation option in copilot
- Feedback (thumbs up/down) on AI messages

### Changed
- Dashboard alerts redesigned with color-coded severity
- Benefits page expanded — click to reveal coverage details

### Fixed
- Keyboard focus not returning to trigger after modal close

---

## [1.3.0] — November 2024
### Added
- Benefits page with deductible/OOP progress tracking
- Coverage detail accordion per benefit type
- Appointments page with upcoming/past tabs

### Changed
- Appointment booking CTA renamed for clarity
- Claim filing button moved above the fold

---

## [1.2.0] — October 2024
### Added
- Design token system (CSS custom properties)
- Skip-to-main-content accessibility link
- Skeleton loading states for dashboard cards

### Changed
- Color palette updated for WCAG 2.1 AA compliance
- Sidebar redesigned with active state indicators

---

## [1.1.0] — September 2024
### Added
- Member dashboard with health activity chart
- Stat cards with trend indicators
- Navigation sidebar

---

## [1.0.0] — August 2024
### Added
- Initial project structure
- Design system foundations
- Component library: Card, Button, Badge, StatCard, ProgressBar
